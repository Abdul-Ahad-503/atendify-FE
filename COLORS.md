## Usage Examples

### Buttons

- **Primary Button**: Background `#2563EB`, Text `#FFFFFF`
- **Secondary Button**: Border `#2563EB`, Text `#2563EB`

### Cards

- Background: `#FFFFFF`
- Border Radius: `12px`
- Shadow: `0px 10px 25px rgba(0, 0, 0, 0.08)`

### Inputs

- Background: `#FFFFFF`
- Border: `#E2E8F0`
- Focus Border: `#2563EB`
- Border Radius: `8px`

### Typography Sizes

- H1: `32px` / SemiBold
- H2: `24px` / SemiBold
- H3: `20px` / Medium
- Body: `16px` / Regular
- Small: `14px` / Regular
- Extra Small: `12px` / Regular
